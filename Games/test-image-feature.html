üìù How to Use

For Multiple Choice Questions: 
{
  "type": "multi",
  "text": "What animal is shown in this GIF?",
  "options": ["Cat", "Dog", "Rabbit", "Bird"],
  "correct": [0],
  "image": "question-images/cat-animation.gif"
}
For Single Answer Questions: 
{
  "type": "single",
  "text": "What color is this object?",
  "answer": "Blue",
  "image": "question-images/blue-object.jpg"
}
For Icebreak Questions: 
{
  "type": "icebreak",
  "text": "What's happening in this scene?",
  "image": "question-images/scene.png"
}
Note: The image field is optional. If omitted or set to null, questions display normally without images.
